# 🧘 Vipassana — Sistema de Insumos

App para gestionar los pedidos de insumos del centro de meditación Vipassana.

## Cómo subir a Vercel

1. Sube esta carpeta a GitHub
2. Conecta el repositorio en vercel.com
3. Vercel detecta Vite automáticamente — solo presiona Deploy

## Desarrollo local

```bash
npm install
npm run dev
```
